HTML

what is HTML

HTML stands for HyperText Markup Language. It is the standard markup language used to create web sitesThe and it is used to describe the structure of a Web page. It tells the web browser how to display text, links, images, and other forms of multimedia on a webpage.

Tags

The keywords which is enclosed between to display the data in browser, every tag as its own behaviour.types of tags are paired and unpaired tags.

1.paired tag

The tag which is having closing and opening tag is known as paired tag.
ex:h1-h6,P...

2. unpaired tag

The tag which closes itself is called self closing tag.
ex: br,img...

Elements

The combination of opening, closing tag and with content between the tags is known as elements.

types of elements

1.Block level elements
2.Inline elements
3.Inline block elements
1.Block level elements

The tag which occupies whole width and creates a new line.
ex:div, h1-h6, p....

2.Inline elements

The elements which occupies only required space and does not create a new line, in css we cannot apply width ,height,padding top andbottom, margin top.
ex:span, img, br.

3.Inline block elements

The element which occupies only required space and doesnot create a new line andwe cn apply width, height, margin and padding properties
